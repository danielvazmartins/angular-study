# angular-study
Repositório para criar projetos de estudo utilizando o Angular v2 ou superior

## Preparar o ambiente

- Instalando o Angular CLI globalmente
npm install -g @angular/cli

- Criando um novo projeto
ng new my-app

- Executar o projeto
cd my-app
ng serve
ng serve --open
ng serve --port

- Criar um componente
ng generate component comp1
ng g c comp1
